# Azure Network Security Project

## Overview
Secure Azure network design with VNets, NSGs, Azure Firewall, and identity-based access control.

## What I implemented
- Created a hub-and-spoke VNet design
- Configured Network Security Groups (NSGs)
- Deployed Azure Firewall
- Used Azure Bastion for secure access
- Applied Conditional Access policies

## Technologies
- Azure Virtual Network (VNet)
- Azure Firewall
- NSG
- Bastion
- Azure AD / Entra ID
